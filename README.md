**简洁但不简陋**  
多搜索引擎、多来源搜索建议、搜索历史记录、识别并打开输入的网址、自定义网站捷径、便签与笔记……  
一个搜索起始页该有的功能都有，麻雀虽小五脏俱全。  
  
**外观与个性化**  
美观好用的 Material Design 3 风格的界面  
德芙般丝滑优雅的 3D 非线性动画  
支持无壁纸、本地上传壁纸、在线壁纸  
支持深浅色模式和更换主题色  
支持隐藏logo和页脚信息  
支持关闭部分动画和效果来降低性能压力  
  
**多语言支持**  
支持简体中文、繁体中文、英语、日语  
  
**Simple but not crude**  
Multiple search engines, multi-source search suggestions, search history, identifying and opening input URLs, customizing website shortcuts, notes and notes
A search starting page should have all the necessary functions, even though it is small, it has all the necessary features.   
  
**Appearance and Personalization**  
Beautiful and user-friendly Material Design 3 style interface  
Dove like silky and elegant 3D non-linear animation  
Support no wallpaper, local wallpaper upload, and online wallpaper  
Supports light and dark color modes and changing theme colors  
Support hiding logo and footer information  
Support turning off some animations and effects to reduce performance pressure
  
**Multi language support**  
Supports Simplified Chinese, Traditional Chinese, English, and Japanese
